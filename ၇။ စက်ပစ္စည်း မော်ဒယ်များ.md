# ၇။ စက်ပစ္စည်း မော်ဒယ်များ

## ၇.၁ Instance လိုင်းများနှင့် .model လိုင်းများ
ngspice netlist သို့ စက်ပစ္စည်းတစ်ခု ထည့်သွင်းရန် လိုင်းနှစ်ကြောင်း လိုအပ်သည်- instance line နှင့် .model line တို့ဖြစ်သည်။

**Instance line:** `XXXXXXX node1 node2 node3 modelname <instpar1=val> <instpar2=val> <off>`

**_.model line:_** `.model modelname modeltype mpar1=val mpar2=val`

Instance line ၏ ပထမအက္ခရာသည် စက်ပစ္စည်းအမျိုးအစားကို ရွေးချယ်သည်။ QXXXXXX သည် ထူးခြားသော အမည်တစ်ခုဖြစ်သည်။ ထို့နောက် စက်ပစ္စည်း node များ၊ modelname နှင့် instance parameters များ (စိတ်ကြိုက်) လိုက်ပါသည်။ .model line သည် model parameters အစုတစ်ခုကို ထည့်သွင်းသည်။ modelname သည် စက်ပစ္စည်းများကို ဤ parameter အစုနှင့် ချိတ်ဆက်ပေးပြီး၊ modeltype သည် မော်ဒယ်အမျိုးအစား (ဥပမာ NPN၊ NMOS) ကို သတ်မှတ်သည်။ Level နှင့် version parameters များက မော်ဒယ်ခွဲများကို ဝင်ရောက်ခွင့်ပြုသည်။

## ၇.၂ Junction Diode များ
**ယေဘူယျပုံစံ:** `DXXXXXXX n+ n- mname <area=val> <m=val> <pj=val> <off> + <ic=vd> <temp=val> <dtemp=val>`

ngspice ရှိ pn junction (diode) သည် SPICE3f5 ထက် ပိုမိုချဲ့ထွင်ထားပြီး perimeter effects၊ high injection level နှင့် temperature dependence တို့ကို ထည့်သွင်းထားသည်။ n+ နှင့် n- သည် positive (anode) နှင့် negative (cathode) nodes များဖြစ်သည်။ mname သည် model name ဖြစ်သည်။ Instance parameters များသည် သက်ဆိုင်ရာလိုင်းပေါ်ရှိ diode အတွက်သာ အကျုံးဝင်သည်။ area သည် saturation current ကို scale လုပ်ပြီး၊ pj သည် sidewall saturation current နှင့် capacitance ကို scale လုပ်သည်။ m သည် area နှင့် perimeter အတွက် multiplier ဖြစ်သည်။

ခေတ်မီ process design kits (PDK) များ၏ လိုအပ်ချက်များကို ဖြည့်ဆည်းရန် LEVEL=3 model parameter သို့မဟုတ် element parameters (lm, wm, lp, wp) များဖြင့် parasitic effects များ (sidewall junction၊ tunnel currents၊ overlap capacitances) ကို မော်ဒယ်လုပ်နိုင်သည်။ OpenVAF/OSDI နည်းလမ်းဖြင့် JUNCAP ကဲ့သို့သော ခေတ်မီ diode မော်ဒယ်များကိုလည်း အသုံးပြုနိုင်သည်။

### ၇.၂.၁ Diode Model (D)
`.model mname type(pname1=pval1 ...)` ပုံစံဖြင့် ဖော်ပြသည်။ ဥပမာ- `.model DMOD D` သည် default parameters များကိုသုံးသည်။ DC characteristics များကို IS, N, RS တို့ဖြင့် ဆုံးဖြတ်သည်။ Charge storage ကို TT နှင့် CJ0, VJ, M တို့ဖြင့် မော်ဒယ်လုပ်သည်။ Temperature dependence ကို EG, XTI, TNOM တို့ဖြင့် ဆုံးဖြတ်သည်။ Reverse breakdown ကို BV နှင့် IBV တို့ဖြင့် မော်ဒယ်လုပ်သည်။

**Junction DC parameters:** IS, JSW, N, RS, BV, IBV, NBV, IKF, IKR, JTUN, JTUNSW, NTUN, XTITUN, ISR, NR

**Junction capacitance parameters:** CJO, CJP, FC, M, MJSW, VJ, PHP, TT

**Metal and Polysilicon Overlap Capacitances (level=3):** LM, LP, WM, WP, XOM, XOI, XM, XP, XW

**Temperature effects:** EG, GAP1, GAP2, TNOM, TRS1, TRS2, TM1, TM2, TTT1, TTT2, XTI, TLEV, TLEVC, CTA, CTP, TCV

**Noise modeling:** KF, AF

### ၇.၂.၂ Diode ညီမျှခြင်းများ
Diode ကို Forward bias၊ Reverse bias နှင့် Breakdown ဟူ၍ နယ်ပယ်သုံးခုဖြင့် မော်ဒယ်လုပ်သည်။ Model parameters များကို area, pj, m တို့ဖြင့် scale လုပ်သည်။ DC, Transient နှင့် AC model equations များတွင် bottom နှင့် sidewall components များအတွက် current များပါဝင်သည်။ Breakdown current ကို တွက်ချက်ရာတွင် BV နှင့် IBV ကိုအသုံးပြုပြီး ဒေသနှစ်ခုကြား ကိုက်ညီမှုရှိစေရန် ချိန်ညှိသည်။ High injection effects များကို IK နှင့် IKR ဖြင့် မော်ဒယ်လုပ်သည်။

Diode capacitance တွင် Depletion capacitance (bottom-wall နှင့် sidewall) နှင့် Diffusion capacitance (TT ဖြင့်မော်ဒယ်လုပ်) တို့ပါဝင်သည်။

**Temperature dependence:** Band-gap energy (EG)၊ leakage current (IS, JSW)၊ contact potentials (VJ, PHP)၊ depletion capacitances (CJ, CJSW)၊ transit time (TT)၊ junction grading coefficient (MJ)၊ series resistance (RS) တို့အတွက် temperature သက်ရောက်မှုများကို ဖော်မြူလာများဖြင့် တွက်ချက်သည်။

**Noise model:** Parasitic resistance RS မှ thermal noise၊ pn junction မှ shot နှင့် flicker noise (KF, AF) တို့ကို မော်ဒယ်လုပ်သည်။

**Self Heating model:** Dissipated power ကို RC parallel circuit သို့ current အဖြစ်ပို့ကာ junction overtemperature ကို voltage အဖြစ်ရယူပြီး electrothermal feedback ဖြင့် current နှင့် capacitance ကိုပြောင်းလဲသည်။

### ၇.၂.၃ OpenVAF/OSDI မှတစ်ဆင့် ရရှိနိုင်သော Diode ပုံစံများ
- **JUNCAP2:** Philips research မှစတင်ခဲ့ပြီး ပေါင်းစည်းဆားကစ်ဒီဇိုင်းတွင် တွင်ကျယ်စွာသုံးသော diode မော်ဒယ်။
- **DIODE_CMC:** JUNCAP2 ကို ပိုမိုကောင်းမွန်အောင်ပြုလုပ်ထားပြီး Series resistor, Diffusion cap (soft recovery), Breakdown voltage temp. coeff., Noise, Min-max parameters များပါဝင်သည်။

## ၇.၃ BJT (Bipolar Junction Transistors)
**ယေဘူယျပုံစံ:** `QXXXXXXX nc nb ne <ns> <tj> mname <area=val> <areac=val> + <areab=val> <m=val> <off> <ic=ve, vce> <temp=val> <dtemp=val>`

nc, nb, ne သည် collector, base, emitter nodes များဖြစ်သည်။ ns သည် (စိတ်ကြိုက်) substrate node၊ tj သည် VBIC မော်ဒယ်အတွက် (စိတ်ကြိုက်) junction temperature node ဖြစ်သည်။ area, areab, areac တို့သည် area factors များဖြစ်သည်။

### ၇.၃.၂ BJT Models (NPN/PNP)
Ngspice သည် BJT မော်ဒယ်သုံးမျိုး ပေးသည်- LEVEL=1 (original SPICE BJT), LEVEL=4 (Advanced VBIC), LEVEL=8 (HICUM/L2)။ OpenVAF/OSDI မှတစ်ဆင့် VBIC, Mextram, HICUM စသည့် Verilog-A မော်ဒယ်များကိုလည်း ရရှိနိုင်သည်။

### ၇.၃.၃ Gummel-Poon Models
Gummel နှင့် Poon တို့၏ integral charge control model ကို လိုက်လျောညီထွေဖြစ်အောင် ပြုလုပ်ထားသည်။ High bias levels များတွင် effects များထည့်သွင်းရန် မူရင်းမော်ဒယ်ကို တိုးချဲ့ထားသည်။ DC model ကို IS, BF, NF, ISE, IKF, NE (forward)၊ IS, BR, NR, ISC, IKR, NC (reverse)၊ VAF, VAR (output conductance) တို့ဖြင့် သတ်မှတ်သည်။

Quasi-Saturation Model extension: RCO, VO, GAMMA, QCO တို့ဖြင့် အသက်သွင်းနိုင်သည်။

**Parameters:** IS, BF, NF, VAF, IKF, ISE, NE, BR, NR, VAR, IKR, ISC, NC, RB, IRB, RBM, RE, RC, CJE, VJE, MJE, TF, CJC, VJC, MJC, CJS, VJS, MJS, XTI, EG, KF, AF, etc. Temperature coefficients များစွာလည်းပါဝင်သည်။

### ၇.၃.၄ VBIC Model
Standard Gummel-Poon (SGP) model မှ တိုးတက်ပြောင်းလဲထားသော မော်ဒယ်ဖြစ်ပြီး ခေတ်မီပေါင်းစည်းဆားကစ် transistors များ၊ discrete switching နှင့် RF transistors များအတွက် သင့်လျော်သည်။ Parasitic substrate transistor၊ Weak avalanche၊ Improved Early effect modelling၊ Self-heating modelling စသည်တို့ကို ထောက်ပံ့သည်။ Self-heating အတွက် simple 1-pole thermal network (RTH, CTH) ကို အသုံးပြုသည်။

### ၇.၃.၅ HICUM level 2 Model
ရူပဗေဒအခြေပြု HIgh-Current Model (HICUM) Level2 ဖြစ်ပြီး SiGe HBTs နှင့် InP HBTs များအတွက် standard compact model တစ်ခုဖြစ်သည်။ Substrate transistor, avalanche effect, physics-based transfer current model, self-heating, excess phase စသည်တို့ကို ထောက်ပံ့သည်။ C, B, E, S, T node များပါဝင်သည်။

### ၇.၃.၆ OpenVAF/OSDI မှတစ်ဆင့် ရရှိနိုင်သော BJT ပုံစံများ
- **HICUM level 0:** ပိုမိုကြီးမားသော ဆားကစ်များအတွက် simulation time ကို လျှော့ချရန် ရည်ရွယ်သည်။
- **HICUM level 2:** HICUM/L2 (အထက်တွင်ကြည့်ပါ)။
- **MEXTRAM 504 and 505:** advanced compact model ဖြစ်ပြီး double-poly၊ SiGe၊ high-voltage power devices များအတွက် သုံးနိုင်သည်။

## ၇.၄ JFET များ
**ယေဘူယျပုံစံ:** `JXXXXXX nd ng ns mname <area> <off> <ic=vds, vgs> <temp=t>`
nd, ng, ns သည် drain, gate, source nodes များဖြစ်သည်။

### ၇.၄.၂-၇.၄.၅ JFET Models
- **Level 1 (Shichman and Hodges):** VTO, BETA, LAMBDA, IS, RD, RS တို့ဖြင့် သတ်မှတ်သည်။ Parker Skellern modification (B parameter) ပါဝင်သည်။
- **Level 2 (Parker Skellern):** ပိုမိုအသေးစိတ်သော မော်ဒယ်ဖြစ်ပြီး parameters များစွာပါဝင်သည်။
- **Temperature model alternatives:** VTO, BETA အတွက် ရွေးချယ်စရာများရှိသည်။
- **Noise model:** NLEV=3 ဖြင့် channel thermal noise အတွက် ပိုမိုတိကျသော ဖော်ပြချက်ကို ရရှိနိုင်သည်။

## ၇.၅ MESFET များ
**ယေဘူယျပုံစံ:** `ZXXXXXXX ND NG NS MNAME <AREA> <OFF> <IC=VDS, VGS>`

### ၇.၅.၂-၇.၅.၅ MESFET Models
- **Level 1 (Statz e.a.):** GaAs FET model ကိုအခြေခံသည်။ VTO, B, BETA, ALPHA, LAMBDA, RD, RS, CGS, CGD, PB တို့ဖြင့် သတ်မှတ်သည်။
- **Level 2-4 (Ytterdal e.a.):** Unified GaAs MESFET Model။
- **hfet1 (level 5) နှင့် hfet2 (level 6):** Heterostructure Field Effect Transistor မော်ဒယ်များ။ hfet2 သည် level 1 ကို ရိုးရှင်းအောင်ပြုလုပ်ထားသော မော်ဒယ်ဖြစ်သည်။

## ၇.၆ MOSFET များ
SPICE3f5 ၏ မူရင်း MOSFET မော်ဒယ်အားလုံးနှင့် open-source အဖြစ်ထုတ်ဝေထားသော မော်ဒယ်သစ်အများစုကို ထောက်ပံ့သည်။

### ၇.၆.၁ MOSFET စက်ပစ္စည်းများ
**ယေဘူယျပုံစံ:** `MXXXXXXX nd ng ns nb mname <m=val> <l=val> <w=val> ...`
nd, ng, ns, nb သည် drain, gate, source, bulk nodes များဖြစ်သည်။ m သည် multiplicity parameter ဖြစ်သည်။ W, L, AD, AS, PD, PS, NRD, NRS စသည့် geometry parameters များကို သတ်မှတ်နိုင်သည်။ BSIM3/BSIM4/BSIMSOI တို့အတွက် local mismatch နှင့် NBTI modeling အတွက် `delvto` နှင့် `mulu0` တို့ကိုလည်း ထောက်ပံ့သည်။

### ၇.၆.၂ MOSFET မော်ဒယ်များ (NMOS/PMOS)
`.model MOSN NMOS level=8 version=3.3.0` ကဲ့သို့သော .model card ဖြင့် invoke လုပ်သည်။ အဓိက မော်ဒယ်များမှာ-
- **MOS Level 1:** Shichman-Hodges model၊ ရှည်လျားသော channel များအတွက်သာ သင့်လျော်။
- **MOS Level 2:** Short-channel effects များကို ဖြေရှင်းရန် ကြိုးပမ်းထားသော်လည်း convergence ပြဿနာများရှိနိုင်သည်။
- **MOS Level 3:** Semi-empirical model၊ တစ်ချိန်က digital design အတွက် တွင်ကျယ်စွာသုံးခဲ့။
- **MOS Level 6:** Short-channel MOSFETs များအတွက် သင့်လျော်ပြီး အကဲဖြတ်ချိန် မြန်ဆန်သည်။
- **MOS Level 9:** Modified Level 3 model (documentation မရှိ)။
- **BSIM1 (level 4) နှင့် BSIM2 (level 5):** Berkeley Short-Channel IGFET Models (ယခု အသုံးမပြုတော့ပါ)။
- **BSIM3 (levels 8, 49) နှင့် BSIM4 (levels 14, 54):** Submicron နှင့် deep-submicron IC ဒီဇိုင်းအတွက် လက်ရှိ industry standards များဖြစ်သည်။
- **BSIMSOI models (levels 10, 58, 55, 56, 57):** Silicon-On-Insulator ဆားကစ်များအတွက် မော်ဒယ်များ။
- **HiSIM models (level 68, 73):** Hiroshima University မှ surface-potential အခြေပြု မော်ဒယ်များ။
- **OpenVAF/OSDI Models:** PSP, BSIM-BULK, BSIM-CMG, EKV3 စသည့် Verilog-A မော်ဒယ်များ။

## ၇.၇ Power MOSFET model (VDMOS)
3 terminals (drain, gate, source) ပါသော ရိုးရှင်းသည့် power MOS model ဖြစ်သည်။ MOS1 model ကို ပြုပြင်ထားသည်။ Gate-source capacitance (CGS)၊ drain-source capacitance (CGDMIN, CGDMAX, A, CJO, FC, M)၊ subthreshold current (KSUBTHRES)၊ quasi-saturation (RQ, VQ) တို့ကို မော်ဒယ်လုပ်သည်။ **Electro-thermal model:** TJ နှင့် TCASE node များပါဝင်ပြီး self-heating effect ကို မော်ဒယ်လုပ်သည် (RTHJC, CTHJ)။
