# Ngspice-Simulator-MM-

**"Ngspice User's Manual Version 44"** အကြောင်းအနှစ်ချုပ် 

### **Ngspice ဆိုတာဘာလဲ?**
[cite_start]Ngspice ဟာ open-source (အခမဲ့ အသုံးပြုနိုင်သော) mixed-level/mixed-signal circuit simulator တစ်ခု ဖြစ်ပါတယ် [cite: 36, 1]။ [cite_start]၎င်းတွင် circuit များကို တွက်ချက်စမ်းသပ်ပေးသည့် simulator နှင့် ရရှိလာသော အချက်အလက်များကို ခွဲခြမ်းစိတ်ဖြာခြင်း (analysis) နှင့် ပုံဖော်ခြင်း (plotting) တို့အတွက် front-end interface တစ်ခု ပါဝင်ပါတယ် [cite: 9]။

### **ဒီလက်စွဲစာအုပ် (Manual) အကြောင်း**
ယခုစာအုပ်သည် ၂၀၂၄ ခုနှစ်၊ [cite_start]ဒီဇင်ဘာလတွင် ထုတ်ဝေခဲ့သော Ngspice Version 44 အတွက် ဖြစ်ပြီး [cite: 21, 24]၊ [cite_start]Ngspice ၏ လုပ်ဆောင်ချက်များ၊ features များ၊ commands များနှင့် အသုံးပြုပုံ အဆင့်ဆင့်ကို အသေးစိတ် ဖော်ပြထားပါတယ် [cite: 36]။ [cite_start]ဤစာအုပ်သည် SPICE အသုံးပြုပုံကို အခြေခံမှစ၍ သင်ကြားပေးသည့် စာအုပ်မဟုတ်သော်လည်း စတင်အသုံးပြုသူများအတွက် အထောက်အကူပြုမည့် အချက်အလက်များ (ဥပမာ - အခန်း ၁၇.၁ တွင် circuit အသေးစားလေးများကို စမ်းသပ်ပုံ) ပါဝင်ပါတယ် [cite: 36]။

### **ပါဝင်သော အဓိက အကြောင်းအရာများ**
* **Simulation အမျိုးအစားများ:** Analog simulation (ဥပမာ - DC, AC, Transient analysis)၊ [cite_start]Digital simulation နှင့် Mixed-signal simulation များအကြောင်း ပါဝင်ပါတယ် [cite: 3]။
* [cite_start]**Circuit Elements နှင့် Models:** Resistors, Capacitors, Diodes, Transistors (BJT), MOSFETs နှင့် အခြား electronic components များအကြောင်းကို အသေးစိတ် ဖော်ပြထားပါတယ် [cite: 5, 12]။
* [cite_start]**အသုံးပြုပုံစံများ:** Netlist ဖိုင်များကို အသုံးပြု၍ simulator ကို batch mode (အမိန့်စာကြောင်းဖြင့် ခိုင်းစေခြင်း)၊ interactive mode (အပြန်အလှန် အမိန့်ပေးခြင်း) သို့မဟုတ် control mode များဖြင့် မည်သို့ အသုံးပြုရမည်ကို ရှင်းပြထားပါတယ် [cite: 4, 9]။
* [cite_start]**စနစ်များ:** Windows, Linux နှင့် macOS တို့တွင် တပ်ဆင် အသုံးပြုပုံများကို ဖော်ပြထားပါတယ် [cite: 9]။
